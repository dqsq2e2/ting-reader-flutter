# Changelog

## 1.0.1 - 2026-06-20

- 修复 Windows 桌面端中文字体渲染发虚、同词字重不一致的问题，随 Windows 发行包加载静态 Noto Sans CJK SC 字体。
- 统一客户端普通文本与控件文案的字重策略，减少正文、按钮、副标题等位置的视觉粗细跳变。
- 修复深色模式下“我的”页面统计数字颜色仍按浅色主题显示的问题。
- “我的下载”页面增加返回入口，保持与其他二级页面一致。
- iOS 发布产物改为标准 IPA 结构，可通过 AltStore、SideStore 等工具自行签名安装。
- 修复 iOS 锁屏播放器首次播放时未显示章节标题、作者和书籍封面的问题。

## 2026-06-18

- 初始化独立 Flutter 客户端仓库资料。
- 补充全平台构建工作流和固定 Release 产物命名。
- 增加 Linux `just_audio_media_kit` 播放后端。
- 补齐 macOS Release 网络客户端 entitlement。
- 记录各平台播放器、后台媒体控制和下载能力边界。
- 关于界面支持检查客户端最新版本并打开下载/安装流程。
- 官网链接和插件仓库链接支持跳转系统浏览器。
